# Agents Workspace

Use this folder only for spawned-agent task artifacts.

Rules
- Write spawned-agent plans here as `YYYY-MM-DD-<topic>-plan.md`.
- Write spawned-agent investigations here as `YYYY-MM-DD-<topic>-investigation.md`.
- Use logical kebab-case topics.
- If multiple files of the same kind exist for one topic on one day, append `-02`, `-03`, etc.
- Do not treat this folder as canonical durable memory.
- Promote durable findings into `/.MEMORY/**` when they matter beyond the current task.
